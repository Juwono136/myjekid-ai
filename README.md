**MyJekID** adalah sebuah sistem aplikasi untuk manajemen layanan kurir atau jasa pesan-antar yang berlokasi di daerah Pulau Sumbawa, Nusa Tenggara Barat. 
Sistem ini dilengkapi dengan **Admin Dashboard** untuk pemantauan real-time dari manajemen kurir dan monitoring order, **AI Chatbot** (didukung oleh Gemini/OpenAI) untuk menangani pelanggan yang ingin melakukan order
dan kurir via WhatsApp secara otomatis.

## Fitur Utama
- **AI Chatbot Real-time:** Layanan pelanggan dan kurir otomatis via WhatsApp.
- **Admin Dashboard Web App:** yang memiliki fitur sebagai berikut:
  - **Dashboard Overview:** Untuk melihat atau monitoring order, distribusi order, transaksi order, dan quick access ke n8n workflow, WAHA Dashboard, Storage, dan Chatbot.
  - **Mitra Kurir:** Manajemen kurir seperti menambah, update, dan menghapus kurir.
